# 🏋️ Bench Press Max Calculator

Консольное приложение на C++ для расчёта и отслеживания прогресса максимального жима лёжа (1ПМ).

```
 ___                                     ___
|   |                                   |   |
|   |=====[=== BENCH PRESS MAX ===]=====|   |
|___|                                   |___|
```

## Возможности

- Расчёт разового максимума по 7 формулам (Эпли, Бжицки, Лэндера, Ломбарди, Мэйхью, О'Коннор, Ватана)
- Регистрация и вход с хешированием пароля (SHA-256)
- Сохранение истории тренировок в локальную базу данных SQLite
- Просмотр истории всех тренировок
- Визуализация прогресса в виде графика (gnuplot)
- Расчёт без авторизации для быстрого использования

## Стек

| Компонент | Технология |
|---|---|
| Язык | C++ |
| База данных | SQLite3 |
| Графики | gnuplot |
| Хеширование | OpenSSL (SHA-256) |

## Установка

### Зависимости

macOS (через Homebrew):

```bash
brew install sqlite3 gnuplot openssl
```

### Сборка и запуск

```bash
git clone https://github.com/Damirka228/bench_press_cpp.git
cd bench_press_cpp
make run
```

Или вручную:

```bash
g++ main.cpp calculator/calculator.cpp database/database.cpp diagram/diagram.cpp auth/auth.cpp \
    -o main -lsqlite3 -lssl -lcrypto -I/opt/homebrew/include -L/opt/homebrew/lib
./main
```

## Использование

При запуске доступно главное меню:

```
1. Регистрация
2. Вход
3. Выход
4. Калькулятор жима (без авторизации)
```

После входа в аккаунт:

```
1. Калькулятор + сохранение тренировки
2. История тренировок + диаграмма прогресса
3. Выйти из аккаунта
```

## Формулы расчёта максимума

| Формула | Описание |
|---|---|
| Эпли | `вес × (1 + повторения / 30)` |
| Бжицки | `вес × (36 / (37 - повторения))` |
| Лэндера | `(100 × вес) / (101.3 - 2.67123 × повторения)` |
| Ломбарди | `вес × повторения^0.1` |
| Мэйхью | `(100 × вес) / (52.2 + 41.9 × e^(-0.055 × повторения))` |
| О'Коннор | `вес × (1 + повторения / 40)` |
| Ватана | `(100 × вес) / (48.8 + 53.8 × e^(-0.075 × повторения))` |

Итоговый результат — среднее значение всех семи формул.

## Безопасность

Пароли хранятся в базе данных в виде SHA-256 хешей, а не открытым текстом.
